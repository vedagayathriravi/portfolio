# Portfolio Website

A modern, responsive portfolio website showcasing my professional experience, education, projects, and skills. Designed to be flexible and adaptable for various job roles in data science, machine learning, and software engineering.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Enhanced navigation experience
- **Interactive Elements**: Hover effects and transitions throughout
- **Mobile-Friendly Navigation**: Hamburger menu for mobile devices
- **SEO Optimized**: Semantic HTML structure
- **Fast Loading**: Optimized CSS and JavaScript

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for easy customization)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## Getting Started

1. **Clone or Download** this repository
2. **Open** `index.html` in your web browser
3. That's it! No build process or dependencies required.

## Deployment Options

### Option 1: GitHub Pages (Free & Easy)

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings → Pages
4. Select the main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Free)

1. Sign up at [Netlify](https://www.netlify.com)
2. Drag and drop the portfolio folder to Netlify
3. Your site will be live instantly with a custom URL

### Option 3: Vercel (Free)

1. Sign up at [Vercel](https://vercel.com)
2. Import your GitHub repository or upload files
3. Deploy with one click

### Option 4: Custom Domain

After deploying to any platform above, you can:
- Add a custom domain in the platform settings
- Update DNS records as instructed
- Your portfolio will be accessible via your custom domain

## Customization

### Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;    /* Main brand color */
    --secondary-color: #8b5cf6;  /* Secondary accent */
    /* ... other variables */
}
```

### Content

- Update personal information in `index.html`
- Modify sections, experiences, projects as needed
- Add or remove skills and categories

### Styling

- All styles are in `styles.css`
- Responsive breakpoints are at 768px and 480px
- Easy to modify fonts, spacing, and colors

## Adding to LinkedIn

1. Deploy your portfolio to one of the platforms above
2. Get your portfolio URL
3. On LinkedIn:
   - Go to your profile
   - Click "Add profile section"
   - Select "Featured" → "Links"
   - Add your portfolio URL
   - Or add it to your contact information section
